# Waveshare Modbus RTU / TCP – Home Assistant

![icon](custom_components/waveshare_modbus/icon.png)

## Description
Intégration HACS pour piloter les boîtiers Waveshare Serial Server en Modbus RTU over TCP ou Modbus TCP.

## Matériel supporté
- RS232 / RS485 / RS422 Ethernet
- 1CH, 2CH RS485
- 8CH RS485 industriel

## Installation
Via HACS ou manuelle (voir documentation complète).

## Configuration UI
Un assistant permet de :
- Choisir le modèle (1/2/8 canaux)
- Définir IP et ports
- Générer automatiquement les hubs Modbus

## Configuration YAML
Exemples détaillés pour capteurs, relais, compteurs, variateurs.

## Diagnostics
Page diagnostics intégrée Home Assistant.
